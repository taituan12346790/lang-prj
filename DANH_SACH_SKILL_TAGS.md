## 📚 GRAMMAR SKILLS (Ngữ pháp)

### 1️⃣ VERB FORMS & TENSES (Động từ & Thì)

#### Present Tenses
- **`present_simple`** - Thì hiện tại đơn
  - Example: "He **plays** football every day"
  - Common errors: "He play" → "plays"

- **`present_simple_third_person`** - Ngôi thứ 3 số ít
  - Example: "She **goes** to school"
  - Common errors: "She go" → "goes"

- **`present_continuous`** - Thì hiện tại tiếp diễn
  - Example: "I **am reading** a book"
  - Common errors: "I reading" → "am reading"

- **`present_perfect`** - Thì hiện tại hoàn thành
  - Example: "I **have seen** that movie"
  - Common errors: "I have saw" → "have seen"

#### Past Tenses
- **`past_simple`** / **`past_tense`** - Thì quá khứ đơn
  - Example: "Yesterday I **went** to school"
  - Common errors: "I go" → "went"

- **`past_tense_irregular`** - Động từ bất quy tắc
  - Example: "She **ate** breakfast"
  - Common errors: "She eated" → "ate"

- **`past_continuous`** - Thì quá khứ tiếp diễn
  - Example: "I **was studying** at 8pm"
  - Common errors: "I studying" → "was studying"

#### Future Tenses
- **`future_simple`** / **`will_future`** - Thì tương lai đơn
  - Example: "I **will go** tomorrow"
  - Common errors: "I will goes" → "will go"

- **`going_to_future`** - Tương lai với "going to"
  - Example: "I **am going to** travel"
  - Common errors: "I going to" → "am going to"

---

### 2️⃣ TO BE VERB (Động từ To Be)

- **`to_be_present`** - To be ở hiện tại
  - Example: "I **am**, you **are**, he **is**"
  - Common errors: "I is" → "am"

- **`to_be_past`** - To be ở quá khứ
  - Example: "I **was**, they **were**"
  - Common errors: "They was" → "were"

- **`to_be_agreement`** - Sự hòa hợp với to be
  - Example: "She **is** happy"
  - Common errors: "She am" → "is"

---

### 3️⃣ SUBJECT-VERB AGREEMENT (Sự hòa hợp chủ ngữ - động từ)

- **`subject_verb_agreement`** - Tổng quát
  - Example: "He **likes** ice cream"
  - Common errors: "He like" → "likes"

- **`singular_plural_agreement`** - Số ít/nhiều
  - Example: "The cat **runs**" vs "Cats **run**"
  - Common errors: "Cat run" → "runs"

---

### 4️⃣ THERE IS/ARE

- **`there_is_are`** - There is/are
  - Example: "There **are** three chairs"
  - Common errors: "There is three chairs" → "are"

- **`there_was_were`** - There was/were
  - Example: "There **were** many people"
  - Common errors: "There was people" → "were"

---

### 5️⃣ ARTICLES (Mạo từ)

- **`articles`** - Mạo từ tổng quát
  - Example: "**a** book, **an** apple, **the** sun"

- **`articles_definite`** - Mạo từ xác định (the)
  - Example: "**The** cat is black"
  - Common errors: "Cat is black" → "The cat"

- **`articles_indefinite`** - Mạo từ không xác định (a/an)
  - Example: "**An** apple, **a** dog"
  - Common errors: "A apple" → "An apple"

- **`a_vs_an`** - Phân biệt a/an
  - Example: "**a** car, **an** hour"
  - Common errors: "a hour" → "an hour"

---

### 6️⃣ PREPOSITIONS (Giới từ)

- **`prepositions`** - Giới từ tổng quát
  - Example: "in, on, at, by, with"

- **`prepositions_time`** - Giới từ chỉ thời gian
  - Example: "**at** 5pm, **on** Monday, **in** 2024"
  - Common errors: "in 5pm" → "at 5pm"

- **`prepositions_place`** - Giới từ chỉ nơi chốn
  - Example: "**at** home, **in** the room, **on** the table"
  - Common errors: "in home" → "at home"

---

### 7️⃣ PRONOUNS (Đại từ)

- **`pronouns`** - Đại từ tổng quát
  - Example: "I, you, he, she, it, we, they"

- **`subject_pronouns`** - Đại từ chủ ngữ
  - Example: "**I** am, **he** is"
  - Common errors: "Me am" → "I am"

- **`object_pronouns`** - Đại từ tân ngữ
  - Example: "Give it to **me**"
  - Common errors: "Give it to I" → "me"

- **`possessive_pronouns`** - Đại từ sở hữu
  - Example: "**my** book, **his** car"
  - Common errors: "me book" → "my book"

---

### 8️⃣ MODAL VERBS (Động từ khuyết thiếu)

- **`modal_verbs`** - Modal verbs tổng quát
  - Example: "can, could, may, might, must, should, will, would"

- **`can_could`** - Can/Could
  - Example: "I **can** swim"
  - Common errors: "I can to swim" → "can swim"

- **`must_have_to`** - Must/Have to
  - Example: "You **must** study"
  - Common errors: "You must to study" → "must study"

- **`should_ought_to`** - Should/Ought to
  - Example: "You **should** go"
  - Common errors: "You should to go" → "should go"

---

### 9️⃣ CONDITIONALS (Câu điều kiện)

- **`conditionals`** - Câu điều kiện tổng quát
  - Example: "If I **have** time, I **will go**"

- **`conditional_type_1`** - Điều kiện loại 1
  - Example: "If it **rains**, I **will stay** home"
  - Common errors: "If it will rain" → "rains"

- **`conditional_type_2`** - Điều kiện loại 2
  - Example: "If I **were** rich, I **would buy** a car"
  - Common errors: "If I was" → "were"

---

### 🔟 PASSIVE VOICE (Câu bị động)

- **`passive_voice`** - Câu bị động
  - Example: "The book **was written** by him"
  - Common errors: "The book written" → "was written"

- **`passive_present`** - Bị động hiện tại
  - Example: "English **is spoken** worldwide"

- **`passive_past`** - Bị động quá khứ
  - Example: "The house **was built** in 1990"

---

### 1️⃣1️⃣ WORD ORDER (Trật tự từ)

- **`word_order`** - Trật tự từ tổng quát
  - Example: "I **always eat** breakfast" (not "I eat always")
  - Common errors: "I eat always" → "always eat"

- **`adjective_order`** - Trật tự tính từ
  - Example: "A **beautiful big red** car"
  - Common errors: "A red big beautiful car"

- **`adverb_position`** - Vị trí trạng từ
  - Example: "I **often** go to school"
  - Common errors: "I go often to school"

---

### 1️⃣2️⃣ QUESTIONS (Câu hỏi)

- **`questions`** / **`question_formation`** - Tạo câu hỏi
  - Example: "**Do** you like coffee?"
  - Common errors: "You like coffee?" → "Do you like?"

- **`wh_questions`** - Câu hỏi Wh-
  - Example: "**What** do you want?"
  - Common errors: "What you want?" → "What do you want?"

- **`yes_no_questions`** - Câu hỏi Yes/No
  - Example: "**Are** you happy?"
  - Common errors: "You are happy?" → "Are you happy?"

---

### 1️⃣3️⃣ NEGATION (Phủ định)

- **`negation`** - Câu phủ định
  - Example: "I **do not** like coffee"
  - Common errors: "I not like" → "do not like"

- **`negative_contractions`** - Rút gọn phủ định
  - Example: "**don't**, **isn't**, **won't**"
  - Common errors: "amn't" → "am not"

---

### 1️⃣4️⃣ OTHER GRAMMAR

- **`comparatives`** - So sánh hơn
  - Example: "**bigger**, **more beautiful**"
  - Common errors: "more big" → "bigger"

- **`superlatives`** - So sánh nhất
  - Example: "**the biggest**, **the most beautiful**"
  - Common errors: "the most big" → "the biggest"

- **`quantifiers`** - Lượng từ
  - Example: "**many**, **much**, **few**, **little**"
  - Common errors: "much books" → "many books"

- **`countable_uncountable`** - Đếm được/không đếm được
  - Example: "**a** book (countable), **some** water (uncountable)"
  - Common errors: "a water" → "some water"

---

## 📖 VOCABULARY SKILLS (Từ vựng)

- **`vocabulary`** - Từ vựng tổng quát
  - Example: Sử dụng sai hoàn toàn từ (ý nghĩa khác)
  - Common errors: "I **go** breakfast" → "eat breakfast"

- **`word_choice`** - Lựa chọn từ
  - Example: "make vs do", "say vs tell"
  - Common errors: "do a mistake" → "make a mistake"

- **`collocations`** - Kết hợp từ
  - Example: "**make** a decision" (not "do a decision")

- **`phrasal_verbs`** - Cụm động từ
  - Example: "**get up**, **look after**"
  - Common errors: Dùng sai preposition

---

## 🔤 SPELLING & OTHER (Chính tả & Khác)

- **`spelling`** / **`spelling_error`** - Chính tả
  - Example: "recieve" → "receive"

- **`punctuation`** - Dấu câu
  - Example: "Hello, how are you**?**"

- **`capitalization`** - Viết hoa
  - Example: "**I** live in **Vietnam**"
